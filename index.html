COVID-19 Classification
Using Chest X-ray Scans

Welcome to the COVID-19 Classification System

This system is developed to help the public to identify if the X-Ray scan falls into one of the following four categories: Normal, Lung Opacity, COVID-19, and Viral Pneumonia.

This system works simply by uploading an X-Ray image and the system analysis the image and tells people if it contains COVID or not. This tool is not meant to replace the doctors it can be considered as an initial screening phase to assist medical professionals.
